## 🌟 项目简介

**优创资源导航** 是一款基于 [WebStack](https://github.com/WebStackPage/WebStackPage.github.io) 二次深度定制的极简资源整合平台。本项目旨在为设计师、开发者以及 AI 爱好者提供一个快速、高效的入口，涵盖了从 AI 大模型到 UI 设计素材的全维度资源。

🔗 **预览地址：** [nav.zyku123.com](http://nav.zyku123.com)

---

## ✨ 核心特性

- 🛠️ **宝塔式精密布局**：参考专业管理后台设计，侧边栏固定，结构严丝合缝。
- 💡 **动态交互感**：全站工具卡片集成 **金色呼吸灯效果**，精准聚焦用户焦点。
- 🔍 **智能即时搜索**：顶部集成胶囊式搜索框，支持站内资源秒级实时过滤。
- 📅 **多功能状态栏**：动态显示公告提示、实时日期及 GitHub 联动信息。
- 📱 **响应式适配**：完美兼容 PC、平板与手机端，随时随地开启探索。

---

## 📂 项目结构说明

- `index.html`：网站主体结构（包含搜索逻辑与品牌信息）。
- `assets/`：存放 Logo、Favicon 等核心视觉资产。
- `css/`：包含自定义的呼吸灯动画及对齐修正样式。
- `js/`：驱动实时过滤搜索及动态公告的逻辑代码。
